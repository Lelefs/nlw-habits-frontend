## 🚀 Rodar a aplicação

- Instale as dependências:

  ```bash
    npm install
  ```

- Rode o projeto em ambiente de desenvolvimento:

  ```bash
    npm run dev
  ```

- Para fazer o build:

  ```bash
    npm run build
  ```
